Coursework

  • Traditional Calculus Sequence. This included 3-D Calculus, vector calculus, ordinary and partial differential equations, boundary value problems.
  • Numerical Analysis 1 and 2. Interpolation, numerical integration, gaussian quadrature, matrices, Schur, Frobenious, and Jordan forms.
  • Mathematical Coding Theory. Hamming codes. Golay codes. Weights of codes, Repetition and error-correcting codes.
  • Cryptography. Stopped short of material needing Abstract Algebra knowledge.Quadratic Number Sieve, basic number theory, pollard rho ansd p-1 tests, divisibility, RSA.
  • Computer Science. Two courses in C++ (up to classes and the other up through virtual classes, inheritence, etc.). Tok a course on Data Structures (search algorithms, black/white trees, breadth/depth first, pseudocode, etc.)
  • Linear Programming. The Simplx method, duality, set up of problems.
  • Applied Analysis. Existence and uniqueness issues, ODes and PDEs, unusual spaces, category theory, eigenvalues of problems.
  • Advanced Linear Algebra. Indefinite linear algebra, sparse matrices, condition numbers, positive-definite matrices, structured matrices (tri-diagonal, upper or lower triangular, hessenberg.) )
